# Pure-sarter

Kick starter project with the following stack:

- Typescript
- Babel steup with `@babel/plugin-transform-typescript`
- No framework yet
- Parcel Bundler
- Now 2.0 ready (With `/api` examples)
- Stylus
- CSS with `autoprefixer`
- CSS with `purecss`
- CSS with `tailwindcss` utility class
